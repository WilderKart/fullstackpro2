Guía de Etiquetas y Propiedades de Estilo en HTML y CSS

Etiquetas de Estilo en HTML
Etiqueta / Propiedad Descripción Ejemplo de Uso Cuándo Usarla
Para estilos internos en una sola página.
rel="stylesheet"> Enlaza un archivo externo .css. rel="stylesheet" href="estilos.css"> Para reutilizar estilos en varias páginas.
class Asigna uno o más nombres de clase para aplicar estilos.
class="caja rojo">Contenido
 Cuando necesitas aplicar el mismo estilo a múltiples elementos.
id Identificador único para un elemento.

id="titulo-principal">Título

 Para dar estilo a un solo elemento específico.
style="" Aplica estilos directamente al elemento.

style="color: blue;">Texto

 Solo en casos puntuales, no recomendado como práctica general.
Propiedades Comunes de CSS
Propiedad Función Ejemplo Uso común
color Cambia el color del texto. p {
color: #333;
} 		Dar color a textos.
font-family Define la fuente del texto. body {
font-family: Arial, sans-serif;
} 		Personalizar tipografía.
font-size Tamaño del texto. h1 {
font-size: 16px;
} 		Ajustar tamaño de letras.
background-color Color de fondo. div {
background-color: #f0f0f0;
} 		Dar color al fondo de un elemento.
margin Espacio exterior de un elemento. .caja {
margin: 10px auto;
} 		Centrar o separar bloques.
Selectores CSS Comunes
Selector Ejemplo Uso
Elemento (p, div, etc.) p {
color: blue;
} 		Selecciona todos los elementos

.
Clase (.) .boton {
padding: 10px;
} 		Aplica estilo a todos los elementos con class="boton".
ID (#) #menu {
width: 100%;
} 		Estiliza un único elemento con id="menu".
Atributo ([type="text"]) input[type="text"] {
border: 1px solid gray;
} 		Selecciona elementos por atributo.
Pseudo-clases (:hover, :nth-child, etc.) a:hover {
color: red;
} 		Aplicar estilo en estados específicos.
